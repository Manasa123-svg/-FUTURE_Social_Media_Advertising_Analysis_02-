# FUTURE_Social_Media_Advertising_Analysis_02

**Project Summary:** Social Media Advertising Performance Dashboard

This project showcases an interactive Power BI dashboard designed to analyze and track the overall effectiveness of social media advertising campaigns. The dashboard provides a 360° view of key performance indicators (KPIs) such as total clicks, total spend, impressions, approved conversions, cost metrics, demographic performance, and campaign-level engagement trends.

# Objectives

. To evaluate the performance of social media ads across different age groups, gender, campaign IDs, and interest categories.

. To provide real-time visibility into daily spend patterns and user engagement.

. To help marketing teams optimize targeting and budget allocation for higher ROI.

# Key Insights Displayed in the Dashboard

# Overall Metrics:

# Total Clicks (12K)

# Total Spend (19.62K)

# Total Impressions (79M)

# Approved Conversions (585)

# CPC (1.68).

# Demographic Analysis:

**Age-wise impressions** highlight the 30–34 age group as the most engaged audience with 36M impressions.

**Gender-wise** clicks show significantly higher male engagement.

# Campaign Performance:

**Filters for Campaign IDs** (916, 936, 1178) allow detailed campaign-level tracking.

# Daily Spend Trends:

A line chart visualizes ad spend fluctuations between 17–29 August, helping identify peak spending and performance days.

# Conversions & Spend by Age:

Donut and bar charts show conversion share and spend distribution across age brackets, supporting targeted budget decisions.

# Interest-based Clicks:

A treemap highlights clicks generated from various interest categories, helping refine audience interest targeting.

# Tools & Techniques Used

. Power BI Desktop for dashboard creation

. Data modeling, DAX measures, interactive slicers

. Visuals: Bar charts, line charts, donut charts, treemap, KPIs, interactive filters

. Data Cleaning & Preparation (Power Query)

# Outcome

The dashboard enables stakeholders to quickly identify high-performing audience segments, optimize campaign strategy, and improve advertising ROI. It provides powerful insights that support data-driven marketing decisions and continuous optimization of digital ad performance.
